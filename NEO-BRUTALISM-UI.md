## üéØ What is Neo-Brutalism?

Neo-Brutalism is a bold, modern design style featuring:
- **Thick black borders** (3-5px)
- **Strong shadows** (offset box-shadows)
- **Vibrant colors** (yellow, pink, blue, green, purple)
- **Bold typography** (uppercase, heavy weights)
- **Flat design** with depth through shadows
- **High contrast** for maximum impact
- **Playful interactions** (hover effects, animations)

**Neo-Buttons:**
- 3px solid black borders
- 4px offset box-shadow

- Active: pushes down

**Neo-Inputs:**
- 3px solid black borders
- 4px offset box-shadow

**Neo-Tables:**
- 4px borders

- 2px cell borders
- Hover: yellow background + scale

**Neo-Badges:**

- Uppercase text

## üé® Customization

### Change Colors
Edit the color classes in `layouts/admin.blade.php`:
```css
.bg-neo-yellow { background: #YOUR_COLOR; }
```

### Adjust Borders
Modify border thickness:
```css
.neo-card { border: 4px solid #000; } /* Change 4px */
```

### Tweak Shadows
Adjust shadow offset:
```css
.neo-card { box-shadow: 8px 8px 0 #000; } /* Change 8px */
```

### Font Changes
Update font family:
```css
* { font-family: 'Your Font', sans-serif; }
```
